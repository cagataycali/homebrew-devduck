## Installation

```bash
# Add the tap
brew tap cagataycali/devduck

# Install devduck
brew install devduck

# Start using
devduck "hello world"
```

## Quick Start

```bash
# Interactive mode
devduck

# One-shot query
devduck "what's the weather?"

# MCP mode (for Claude Desktop)
devduck --mcp

# Custom ports
devduck --tcp-port 9000 --ws-port 8080
```
